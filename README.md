# 🌤 Weather App

A responsive and feature-rich Weather Web Application built using **HTML, CSS, and JavaScript**, integrated with the **OpenWeather API**.

## ✨ Features

- 🔍 Search weather by city name
- 📍 Get weather using current location (Geolocation)
- 🎤 Voice search support
- 🌡 Real-time temperature, humidity, wind speed
- ☁ Weather conditions with icons
- 📅 5-day forecast
- 🌓 Dark / Light mode toggle
- 📜 Search history (stored in localStorage)
- 🌈 Animated weather UI (rain, clouds, sun effects)
- 📱 Fully responsive design

## 🛠 Tech Stack

- HTML5
- CSS3 (Glassmorphism UI, Animations)
- JavaScript (ES6+)
- OpenWeather API
- Browser Geolocation API
- Web Speech API
